Kunsthaus

O jogo Kunsthaus é feito em sua grande maioria em P5, que é uma biblioteca de JavaScript que tem como objetivo tornar acessível a codificação para designers e artistas para web, mas a aplicação também conta com CSS para a parte de interface como nos botões e nos inputs (Caixas para digitar texto) e com alguns atributos do HTML como os tittle de elementos (Pequeno texto que aparece ao deixar o mouse parado sobre o elemento).

O elemento chave na programação para tornar todo o jogo possível foi a troca de telas, que foi feita com objetos, basicamente em quase todas as interações temos a troca de tela para mostrar os novos elementos, pois a parte gráfica em sua grande maioria foi feita através de background do tamanho da tela para haver a mudança.
